# Heart Disease Prediction Project

Итоговый проект по теме "Глубокое обучение и нейронные сети" с решением задачи бинарной классификации (предсказание наличия сердечного заболевания).

## Project Structure

- `DL_PJ_heart_disease.ipynb` - чистая версия ноутбука (без сохраненных outputs).
- `DL_PJ_heart_disease.executed.ipynb` - версия ноутбука с выполненными ячейками, графиками и результатами.
- `train.csv` - обучающая выборка.
- `test.csv` - тестовая выборка.
- `submission.csv` - итоговые предсказания в формате соревнования.
- `pytorch_model.pt` - сохраненные веса обученной PyTorch-модели.
- `article-summary.md` - краткое содержание выбранной статьи.
- `Задание и критерии оценивания.md` - условия и критерии оценивания проекта.

## What Was Done

- Проведен анализ и предобработка данных.
- Обучены и сравнены классические ML-модели.
- Реализована и обучена нейросеть на PyTorch.
- Построены метрики и визуализации качества.
- Выполнен инференс на тестовых данных и сформирован `submission.csv`.

## How To View Results

Для просмотра всех результатов, графиков и таблиц открой:

- `DL_PJ_heart_disease.executed.ipynb`

Если нужен только код и структура решения без outputs:

- `DL_PJ_heart_disease.ipynb`

## Requirements

Основные библиотеки, использованные в проекте:

- Python 3.x
- pandas
- numpy

- seaborn

- joblib

## Reproducibility

1. Установить зависимости.
2. Открыть ноутбук `DL_PJ_heart_disease.ipynb`.
3. Запустить ячейки последовательно (Run All).
4. При необходимости сохранить новую версию с outputs.
